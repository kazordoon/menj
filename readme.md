# MENJ

MENJ CLI - Scaffold for API with NodeJS, Express, MongoDB and JWT.

### NPM
[https://www.npmjs.com/package/menj](https://www.npmjs.com/package/menj "https://www.npmjs.com/package/menj")

### Install
- npm install -g menj

### Use
- menj new your-project
